далее в консоле переходим в эту папку /var/tmp/ и выполняем команды

Команда импорта ВМ из файла VMDK:
>- **# qm importdisk <VMID> <source_path> <storage_name>**
```bash
qm importdisk 100 /var/tmp/my-disk.vmdk local-lvm
```
Команда выполняет следующие действия:

- 100: Указывает ID виртуальной машины (VMID), к которой будет привязан диск.
- /var/tmp/my-disk.vmdk: Путь к исходному файлу диска, который вы хотите импортировать.
- local-lvm: Целевое хранилище, где будет размещен новый диск
