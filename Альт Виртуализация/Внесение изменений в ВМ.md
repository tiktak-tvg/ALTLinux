### Внесение изменений в ВМ
Вносить изменения в конфигурацию ВМ можно и после ее создания. Для того чтобы внести изменения в конфигурацию ВМ, необходимо выбрать ВМ и перейти на вкладку Оборудование.<br> 
На этой вкладке следует выбрать ресурс и нажать кнопку ``Редактировать``:
Оборудование ВМ<br> 
> - Примечание
В случаях, когда изменение не может быть выполнено в горячем режиме, оно будет зарегистрировано как ожидающее изменение (выделяется цветом). <br> Такие изменения будут применены только после перезагрузки ВМ.

#### Управление образами виртуальных дисков
Образ виртуального диска является файлом или группой файлов, в которых ВМ хранит свои данные.<br> 
qemu-img — утилита для манипулирования образами дисков машин QEMU. qemu-img позволяет выполнять операции по созданию образов различных форматов, конвертировать файлы-образы между этими форматами, получать информацию об образах и объединять снимки ВМ для тех форматов, которые это поддерживают.<br> 
> - Примеры использования утилиты qemu-img:<br> 
преобразование (конвертация) vmdk-образа виртуального накопителя VMware под названием test в формат qcow2:
```bash
# qemu-img convert -f vmdk test.vmdk -O qcow2 test.qcow2
```
создание образа test в формате RAW, размером 40 ГБ:
```bash
# qemu-img create -f raw test.raw 40G
```
изменение размера виртуального диска:
```bash
# qemu-img resize -f raw test.raw 80G
```
просмотр информации об образе:
```bash
# qemu-img info test.raw
```
Для управления образами виртуальных дисков в веб-интерфейсе PVE необходимо выбрать ВМ и перейти на вкладку Оборудование. <br> После выбора образа диска станут доступными кнопки: Добавить, Отключить, Редактировать, Переместить хранилище, Переназначить владельца, Изменить размер:<br> 
Управление образом виртуального диска
#### Добавление виртуального диска в ВМ
Для добавления образа виртуального диска к ВМ необходимо:<br> 
Перейти на вкладку Оборудование;<br> 
Нажать кнопку Добавить и выбрать в выпадающем списке пункт Жесткий диск:<br> 
Кнопка «Добавить» → «Жесткий диск»<br> 
Указать параметры жесткого диска и нажать кнопку Добавить:<br> 
Опции добавления жесткого диска<br> 
#### Удаление образа виртуального диска
Для удаления образа виртуального диска к ВМ необходимо:<br> 
Перейти на вкладку Оборудование;<br> 
Выбрать образ диска ВМ;<br> 
Нажать кнопку Отключить;<br> 
В окне подтверждения нажать кнопку Да для подтверждения действия. При этом виртуальный диск будет отсоединен от ВМ, но не удален полностью. <br> Он будет присутствовать в списке оборудования ВМ как Неиспользуемый диск:<br> 
Неиспользуемый диск<br> 
Чтобы удалить образ диска окончательно, следует выбрать неиспользуемый диск и нажать кнопку Удалить.<br> 
Если образ диска был отключен от ВМ по ошибке, можно повторно подключить его к ВМ, выполнив следующие действия:<br> 
Выбрать неиспользуемый диск;<br> 
Нажать кнопку Редактировать;<br> 
В открывшемся диалоговом окне изменить, если это необходимо, параметры Шина/Устройство:<br> 
Подключение неиспользуемого диска<br> 
Нажать кнопку Добавить для повторного подключения образа диска.<br> 
#### Изменение размера диска
Функция изменения размера поддерживает только увеличение размера файла образа виртуального диска.<br> 
При изменении размера образа виртуального диска изменяется только размер файла образа виртуального диска. <br> После изменения размера файла, разделы жесткого диска должны быть изменены внутри самой ВМ.<br> 
Для изменения размера виртуального диска необходимо:<br> 
Перейти на вкладку Оборудование;<br> 
Выбрать образ диска ВМ;<br> 
Нажать кнопку Действие над диском → Изменить размер;<br> 
В открывшемся диалоговом окне в поле Увеличение размера (ГиБ) ввести значение, на которое необходимо увеличить размер диска. <br> Например, если размер существующего диска составляет 20 ГиБ для изменения размера диска до 30 ГиБ следует ввести число 10<br> :
- Изменение размера диска<br>
Нажать кнопку Изменить размер диска.<br> 
Команда изменения размера виртуального диска:<br> 
```bash
# qm resize <vm_id> <virtual_disk> [+]<size>
```
> - Примечание
> - Если указать размер диска со знаком «+», то данное значение добавится к реальному размеру тома, без знака «+» указывается абсолютное значение. Уменьшение размера диска не поддерживается.<br>
> Например, изменить размер виртуального диска до 80 ГиБ:
```bash
# qm resize 100 scsi1 80G
```
#### Перемещение диска в другое хранилище
Образы виртуального диска могут перемещаться с одного хранилища на другое в пределах одного кластера.<br>
Для перемещения образа диска необходимо:<br>
Перейти на вкладку Оборудование;<br>
Выбрать образ диска, который необходимо переместить;<br>
Нажать кнопку Действие над диском → Переместить хранилище;<br>
В открывшемся диалоговом окне в выпадающем меню Целевое хранилище выбрать хранилище-получатель, место, куда будет перемещен образ виртуального диска:<br>
Диалоговое окно перемещения диска<br>
В выпадающем меню Формат выбрать формат образа диска. Этот параметр полезен для преобразования образа диска из одного формата в другой;<br>
Отметить, если это необходимо, пункт Удалить источник для удаления образа диска из исходного хранилища после его перемещения в новое хранилище;<br>
Нажать кнопку Переместить диск.<br>
Команда перемещения образа диска в другое хранилище:<br>
```bash
# qm move-disk <vm_id> <virtual_disk> <storage>
```
#### Переназначение диска другой ВМ
При переназначении образа диска другой ВМ, диск будет удалён из исходной ВМ и подключен к целевой ВМ.<br>
Для переназначения образа диска другой ВМ необходимо:<br>
Перейти на вкладку Оборудование;<br>
Выбрать образ диска, который необходимо переназначить;<br>
Нажать кнопку Действие над диском → Переназначить владельца;<br>
В открывшемся диалоговом окне в выпадающем меню Целевая гостевая система выбрать целевую ВМ, место, куда будет перемещен образ виртуального диска:<br>
Диалоговое окно переназначения диска<br>
Выбрать нужные параметры в выпадающем меню Шина/Устройство;<br>
Нажать кнопку Переназначить диск.<br>
Команда переназначения образа диска другой ВМ:
```bash
# qm move-disk <vm_id> <virtual_disk> --target-vmid <vm_id> --target-disk <virtual_disk>
```
Пример удаления образа диска scsi0 из ВМ 107 и подключение его как scsi1 к ВМ 10007:
```bash
# qm move-disk 107 scsi0 --target-vmid 10007 --target-disk scsi1
```
