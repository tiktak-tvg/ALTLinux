### Загрузка системы
Для того чтобы начать установку ОС Альт Виртуализация редакция PVE, достаточно загрузиться с носителя, на котором записан дистрибутив.
> Примечание
> - Предварительно следует включить в BIOS опцию загрузки с оптического привода или с USB-устройства.<br> 
> - В большинстве случаев указание способа входа в BIOS отображается на вашем мониторе непосредственно после включения компьютера.<br> 
> - Способ входа в меню BIOS и информация о расположении настроек определяется производителем используемого оборудования.<br> 
> - За информацией можно обратиться к документации на ваше оборудование.<br>

Загрузка с установочного диска или специально подготовленного USB-flash-накопителя начинается с меню, в котором перечислено несколько вариантов загрузки.<br> 
Кроме установки системы с установочного диска, в данном меню доступны несколько вариантов сетевой установки:<br> 
Install ALT Virtualization PVE 11.0 — установка операционной системы;<br> 
VNC install (edit to set server IP address) — установка по VNC с соединением с устанавливаемой машины на сервер VNC с заданным IP-адресом.<br>  
Параметры установки по VNC передаются как параметры ядра.<br>  
Нажатие клавиши E позволяет задать IP-адрес компьютера, с которого будет происходить управление (для приёма подключения на сервере VNC следует запустить, например, vncviewer --listen):

<img width="1024" height="500" alt="image" src="https://github.com/user-attachments/assets/5e85baca-e71a-47b8-8782-d3556a9ae7b3" />

VNC install (edit to set password and connect here) — установка по VNC с соединением в сторону устанавливаемой машины. Параметры установки по VNC передаются как параметры ядра. 

Нажатие клавиши E позволяет задать пароль (по умолчанию — VNCPWD):

<img width="1024" height="500" alt="image" src="https://github.com/user-attachments/assets/e079f2ed-fbd6-4281-ba7e-ffdce81261ee" />

Rescue LiveCD — восстановление уже установленной, но так или иначе поврежденной ОС Linux путем запуска небольшого образа ОС в оперативной памяти.<br>  
Восстановление системы потребует некоторой квалификации.<br>  
Этот пункт также может быть использован для сбора информации об оборудовании компьютера, которую можно отправить разработчикам, если ОС Альт Виртуализация редакция PVE устанавливается и работает неправильно. <br> 
Загрузка восстановительного режима заканчивается запросом на ввод логина суперпользователя (root):
```
Log in as root
localhost login:
```
Change language (press F2) — позволяет выбрать язык интерфейса загрузчика и программы установки (нажатие клавиши F2 вызывает такое же действие);<br> 
Memory Test (may not work with Secure Boot) — проверка целостности оперативной памяти. <br> 
Процесс диагностики заключается в проведении нескольких этапов тестирования каждого отдельного модуля ОЗУ (данный процесс будет выполняться бесконечно, пока его не остановят, необходимо дождаться окончания хотя бы одного цикла проверки);
UEFI Shell (may not work with Secure Boot) — оболочка/терминал для прошивки, позволяющий запускать EFI-приложения, в том числе загрузчики UEFI;<br> 
UEFI Firmware Settings — позволяет получить доступ к настройкам UEFI.<br>

Пункт Boot from hard drive — позволяет запустить уже установленную на жёсткий диск операционную систему.
> Примечание
Мышь на этом этапе установки не поддерживается.<br> 
Для выбора опций установки и различных вариантов необходимо использовать клавиатуру.<br> 
Нажатием клавиши E можно вызвать редактор параметров текущего пункта загрузки.<br>  
Если система настроена правильно, то редактировать их нет необходимости.<br> 
Чтобы начать процесс установки, нужно клавишами перемещения курсора вверх и вниз выбрать пункт меню Install ALT Virtualization PVE 11.0 и нажать Enter. <br> 
Начальный этап установки не требует вмешательства пользователя: происходит автоматическое определение оборудования и запуск компонентов программы установки.<br>  
Сообщения о происходящем на данном этапе можно просмотреть, нажав клавишу ESC.<br> 
> Примечание
В начальном загрузчике установлено небольшое время ожидания: если в этот момент не предпринимать никаких действий, то будет загружена та система, которая уже установлена на жестком диске. <br> 
Если вы пропустили нужный момент, перезагрузите компьютер и вовремя выберите пункт Install ALT Virtualization PVE 11.0.<br>
